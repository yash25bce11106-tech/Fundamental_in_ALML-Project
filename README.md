#  Smart Anti-Scam Detection System
## Project Overview
The Smart Anti-Scam Detection System is a web-based application developed using Python (Flask) that helps users identify potential online scams such as phishing messages, suspicious URLs, and fraudulent schemes.
This project aims to spread awareness and provide a simple tool to detect scams.
---
##  Problem Statement
Online scams such as UPI fraud, phishing messages, and fake links are increasing rapidly. Many users fall victim due to lack of awareness.
---
##  Objectives
- Detect scam messages using keyword analysis
- Identify suspicious URLs
- Allow users to report scams
- Educate users through a quiz system
---
##  Features
-  Message Scam Checker
-  URL Safety Checker
-  Report Scam Feature
-  Interactive Quiz
---
##  Tech Stack
- Python
- Flask
- HTML
- CSS (Inline)
---
##  How to Run the Project
1. Clone the repository:
   git clone https://github.com/yourusername/anti-scam-project.git
2. Navigate to the folder:
   cd anti-scam-project
3. Install dependencies:
   pip install flask
4. Run the app:
   python app.py
5. Open browser:
   http://127.0.0.1:5000
